# Coreless

Coreless 초대형 단일 방의 40차 완성 빌드입니다.

- 플레이 사이트: https://coreless-mega-room.leesodong.chatgpt.site
- 작업 브랜치: `rebuild/mega-room-40pass`
- 조작: `A/D` 이동, `Space` 점프, `Shift` 대시, `E` 집게, `F` 공격, `R` 재시작, `B` 청사진

사이트에 접속하면 별도 설치나 실행 명령 없이 게임으로 바로 이동합니다. 40차 이후에는 배경 흐림, 형태 단순화, 캐릭터·환경 스타일 통합 보정을 별도 단계로 진행합니다.
