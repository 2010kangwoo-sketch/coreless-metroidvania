# Coreless 새로운 40차 제작 — 2차

## 선행 확인: 1차 실제 화면

1차 코드를 실제 Canvas 렌더링으로 실행해 다음 문제를 확인했다.

- 전체 설계도 글자 크기가 축척과 반대로 계산되어 글자가 방보다 크게 겹침
- 검증 패널이 기본 표시되어 플레이 화면 오른쪽을 크게 가림
- 기본 화면이 실제 게임이 아니라 설계도임
- 이동 시험실이 첫 번째·두 번째 실제 방 구조를 반영하지 않음
- 카메라 경계, 방 연결 통로, 피해, 체크포인트가 아직 없음

## 2차 구현

- 기본 시작 화면을 실제 플레이 화면으로 변경
- 이동 도입 방: 1200×680 고정 카메라, 낮은 장애물 3개
- 정밀 점프 방: 2400×680 추적 카메라, 순차 발판 9개
- 두 방 사이를 아래쪽 통로 하나로만 연결
- 가시 8구간, 피격·넉백·무적 시간 적용
- 시작·중간·종료 체크포인트 3개
- 낙하 및 체력 소진 시 체크포인트 복귀
- 체력 HUD와 대시 쿨타임 HUD를 월드 오브젝트보다 위에 고정
- F1 설계도, F2 검증표는 필요할 때만 표시

## 검증

- `node --check src/main.js`: 통과
- 실제 Canvas 화면 캡처: 시작, 이동 도입 끝, 정밀 점프 시작·중간·끝, 설계도, 검증표 확인
- 실제 물리 입력 완주 시험: 648프레임, 사망 0회, 종료 체크포인트 도달
- 정밀 점프 최대 간격: 80px
- 자동 감사: 12/12 통과
