# Coreless 재제작 40차 — 1차 기준 확정

## 목적

기존 v59-16의 압축 분할 로더와 10개 조각 실행 구조를 새 제작선에서 사용하지 않는다. `rebuild/mega-room-40pass` 브랜치는 하나의 `src/main.js`를 기준 실행본으로 유지한다.

## 확정 단일 루트

1. 이동 도입
2. 정밀 점프
3. 벽점프 수직축
4. 집게 횡단
5. 벽면 터렛 회랑
6. 전투 경기장
7. 장거리 붕괴석 추격
8. 체크포인트 허브
9. 혼합 이동·전투
10. 최종 상승·출구

분기와 선택 경로는 만들지 않는다. 각 구간은 입구 하나와 출구 하나만 가진다.

## 1차 자동 검사

- 10개 공간의 순서가 1~10으로 연속인지 검사
- 단일 좌표열만 존재하는지 검사
- 첫 시작점과 최종 출구점 일치 검사
- 모든 입구·출구가 해당 공간 내부인지 검사
- 전투 공간 최소 폭 검사
- 통로 폭 기준 검사
- 점프 상승량과 대시 거리 범위 검사
- 카메라 단위와 캔버스 크기 일치 검사
- 전체 공간이 월드 경계 내부인지 검사

## 작업 제한

- 자동 수정 재시도는 최대 2회
- 검사 실패 시 무한 반복 금지
- 실패 원인과 미완료 항목을 기록하고 중단
- 실제 기능 분리가 필요하기 전까지 런타임 파일을 임의로 분할하지 않음

## 다음 차수

2차에서는 1~2번 구역의 실제 충돌 지형과 카메라 경계를 구현한다. 최종 그래픽은 아직 적용하지 않고, 통과성과 공간 크기를 먼저 확정한다.
