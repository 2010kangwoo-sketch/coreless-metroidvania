# 새로운 40차 제작 — 8차 복원 재제작

## 목표

이전 8차 수정본이 기능 시험장처럼 축소된 문제를 폐기하고, 수정 전 8차의 넓은 공간감·카메라 분위기·추격 압박을 기준으로 8차를 다시 구성했다. 벽점프, 대시 가시밭, 정밀 파쿠르를 별도 시험 구역으로 나누지 않고 하나의 거대한 하강형 추격 통로에 통합했다.

## 공간 구조

- 월드 크기: 4400×2500px
- 실제 스네이크 경로 길이: 약 17767.26px
- 위에서 아래로 내려가는 4개 주행 레인
- 3개의 완곡한 헤어핀 전환부
- 모든 레인에 오르막, 내리막, 골, 짧은 언덕을 혼합
- 진입 체크포인트: `cp_chase_prep`
- 종료 체크포인트: `cp_snake_chase_entry`

## 추격 및 조작

- 붕괴석은 미리 정한 직선이 아니라 전체 스네이크 곡선의 위치와 기울기를 따라 이동한다.
- 완곡한 전환부에서는 수평 이동에서 하강으로 자연스럽게 이어진 뒤 반대 레인으로 진입한다.
- 형태가 다른 파괴 구조물 9개 그룹을 붕괴석이 파괴하며, 파괴 시 감속과 파편·카메라 흔들림이 발생한다.
- 연속 벽점프가 필요한 수직 챔버를 포함한다.
- 대시 없이는 사망하고 대시 중에는 통과할 수 있는 가시밭 2개를 포함한다. 붕괴석은 가시를 무시한다.
- 5개의 정밀 파쿠르 발판과 낙하 시 사망하는 심연을 배치해 아래 레인으로 떨어지는 우회를 막았다.
- Space 입력 유지 시간에 따라 점프 높이가 달라지는 가변 점프를 적용했다.
- 이동 발판은 탑승 중 플레이어의 상대 위치를 유지한다.

## 실행 중 발견해 수정한 문제

1. 첫 파괴 구조의 교차 보강대가 통로를 완전히 막아 매달린 아치형 구조로 변경했다.
2. 시각적 곡선과 실제 충돌 지형이 분리되어 있던 세 헤어핀에 연결 경사 지형을 추가했다.
3. 높은 구조물 옆면을 스치며 점프할 때 캐릭터가 지형 아래로 순간 이동하는 수직 충돌 판정을 수정했다.
4. 첫 번째와 두 번째 미로벽을 천장형·바닥형 구조가 번갈아 나타나는 형태로 다시 배치했다.
5. 벽점프 챔버 입구를 막던 우측 벽 하단을 들어 올려 진입 후 벽점프가 필수가 되도록 조정했다.
6. 정밀 파쿠르 발판 폭과 마지막 착지 발판을 조정하고 아래에 치명적 심연을 추가했다.
7. 마지막 보루 구조를 통과 가능한 복합 구조로 다시 만들었다.

- 자동 구조 감사: 18/18
