# Coreless 새로운 40차 제작 — 3차

## 선행 확인: 2차 실제 화면

2차 게임을 실제 Canvas와 Chromium에서 실행해 다음 문제를 확인했다.

- 체크포인트와 플레이어가 같은 좌표에 생성되어 서로 겹침
- 하단 안내판이 정밀 점프 구역의 가시와 지형을 가림
- 방 경계에서 카메라가 1200px를 천천히 따라오며 시야가 늦게 정렬됨
- 방 전환 암전이 HUD 위에 그려져 체력과 대시 표시까지 어두워짐

## 3차 수정

- 체크포인트의 표시 좌표와 부활 좌표 분리
- 안내판을 화면 왼쪽 상단으로 이동하고 일정 시간 뒤 숨김
- 방 진입 시 카메라를 방 시작점에 즉시 정렬
- 전환 암전을 월드와 HUD 사이에 배치해 HUD 가독성 유지
- 기존 이동 도입·정밀 점프 구간의 단일 경로와 물리 수치 유지

## 벽점프 수직축 구현

- 세 번째 공간을 x=3600 이후의 실제 수직 공간으로 연결
- 세로 월드 범위: y=-900~680
- 벽 사이 실제 통과 폭: 200px
- 수직 벽 높이: 1220px
- 벽 미끄러짐 속도: 3.15px/frame
- 벽점프 상승량: 약 102.4px
- 교대로 배치된 일방통행 휴식 발판 5개
- 입구와 상단 체크포인트
- 플레이어 높이를 따라가는 수직 카메라
- 상단 출구로 이어지는 마지막 벽점프

## 실제 플레이 검증

### 전체 1~3구역

- 총 1323프레임
- 사망 0회
- 정밀 점프 11회
- 벽점프 18회
- 벽 미끄러짐 판정 12회
- 최종 체크포인트 `cp_shaft_top` 도달

### 실제 Chromium

- 실제 D키 입력으로 캐릭터 x=126 → 약 235.76 이동 확인
- 벽점프 수직축 614프레임 완주
- 벽점프 18회, 벽 미끄러짐 13회
- 실제 브라우저 콘솔 오류 0건
- 자동 감사 12/12 통과
