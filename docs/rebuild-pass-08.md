# Coreless 새로운 40차 제작 — 8차 재제작

## 재제작 이유

기존 8차는 긴 일자 경사에서 점프와 대시를 반복하는 구조라서, 실제 체감이 단순한 러닝 게임에 가까웠다. 이에 기존 8차를 실행 대상에서 제외하고, 아래로 꺾이며 진행하는 스네이크형 추격 훈련 구간으로 전면 교체했다. 기존 9차도 현재 `index.html`에서 불러오지 않으므로, 새 9차가 완성될 때까지 실행되지 않는다.

## 7차 연결

- 7차 종료 체크포인트 `cp_mixed_end` 또는 기존 월드 x좌표 14500px 이상에서 자동 활성화
- 수동 검사 키: F8
- 8차가 시작되면 기존 캔버스를 교체해 이전 화면과 입력이 겹치지 않게 처리
- 시작 체크포인트: `cp_snake_prep`
- 종료 체크포인트: `cp_snake_chase_entry`

## 스네이크형 단일 경로

- 내부 월드 크기: 2700×1660px
- 실제 경로 길이: 약 6228.40px
- 진행 순서: 상단 오른쪽 이동 → 첫 수직 하강 → 중단 왼쪽 이동 → 두 번째 수직 하강 → 하단 오른쪽 이동
- 수평 방향 전환: 2회
- 수직 하강: 2회
- 갈림길과 우회 바닥 없음
- 바닥 화살표와 구간 이름으로 다음 방향을 명확히 표시

## 파괴 가능한 미로 구조물

작은 돌무더기만 반복하지 않고, 서로 다른 높이와 틈을 가진 구조물을 8개 그룹, 총 21개 조각으로 구성했다.

- 상단 지그재그 파괴문 3개 그룹
- 첫 번째 하강 통로의 교차 리브 구조
- 중단의 상·하단 틈 파괴문
- 벽점프 굴뚝 내부의 격자 구조
- 두 번째 하강 통로 리브 구조
- 하단 파괴문 2개 그룹과 최종 보강벽

플레이어는 구조물의 틈을 점프, 하강, 벽점프로 통과한다. 붕괴석은 구조물을 무시하지 않고 실제 경로 진행값이 구조물 위치에 닿았을 때 같은 그룹 전체를 파괴한다. 한 그룹을 파괴할 때마다 돌의 속도가 1.05px/frame 감소하고 34프레임 동안 감속한다.

## 조작 요소

### 1. 지그재그 파괴문

상단 통로에 상·하단이 번갈아 막힌 구조물을 배치했다. 플레이어는 같은 높이의 장애물을 반복해서 넘는 것이 아니라, 낮은 점프와 높은 점프의 타이밍을 바꿔야 한다.

### 2. 좁은 수직 하강

첫 번째와 두 번째 하강 통로에는 좌우에서 번갈아 돌출된 리브를 배치했다. 곧바로 아래로 떨어지는 것이 아니라, 공중 방향 조절과 벽 슬라이드를 사용해 빈 공간으로 내려가야 한다.

### 3. 연속 벽점프 굴뚝

중단에는 폭 112px의 굴뚝과 높이 175px의 중앙 석벽을 배치했다. 오른쪽에서 굴뚝으로 들어간 뒤 좌우 벽을 번갈아 차며 올라가 석벽 위를 넘어야 한다. 벽에 닿은 상태에서 하강 속도는 2.4px/frame으로 제한되고, 벽점프 속도는 수평 6.4px/frame, 수직 10.8px/frame이다.

### 4. 이동 발판과 대시 간격

하단 경로 중앙에는 260px의 바닥 공백과 좌우로 왕복하는 폭 156px 이동 발판을 배치했다. 발판 위에 서 있으면 발판의 프레임별 이동량이 플레이어에게 그대로 전달된다. 이동 발판에서 점프한 뒤 대시로 다음 바닥에 도달하도록 구성했다.

## 붕괴석 규칙

- 플레이어 크기: 34×48px
- 붕괴석 반지름: 68px
- 붕괴석 기본 속도: 3.05px/frame
- 최대 속도: 5.45px/frame
- 목표 간격: 430px
- 압사 또는 추락 후 복귀: 42프레임

붕괴석은 시작 직후 바로 떨어지지 않는다. 플레이어가 상단 미로 구조의 대부분을 지나 경로 진행값 1250px에 도달하면 120프레임 경고가 시작되고, 이후 돌이 추격을 시작한다. 연습 요소를 읽기도 전에 돌에 따라잡히는 기존 문제를 줄였다.

카메라는 평상시 플레이어를 따라가고, 돌이 낙하한 뒤에는 플레이어와 붕괴석의 중간을 우선해 두 대상을 같은 화면에 유지한다.

## Chromium 기능 검사

전체 구간을 무중단으로 자동 완주했다고 보고하지 않는다. 8차 재제작에서는 서로 다른 조작 요소를 실제 키보드 이벤트와 결정적 테스트 기능으로 나누어 검사했다.

### 실제 키보드 검사

- 상단 통로에서 실제 D와 Space 입력으로 지그재그 구조물 구간 이동 확인
- 굴뚝 내부에서 실제 A·D·Space 교대 입력으로 벽점프 4회 확인
- 벽점프 검사 시작 y좌표 약 850px, 종료 y좌표 약 674.58px
- 종료 지점 앞에서 실제 D 입력으로 `cp_snake_chase_entry` 활성화 확인

### 이동 발판 검사

- 이동 발판 위에 플레이어를 배치하고 120프레임 진행
- 플레이어와 발판의 상대 x좌표: 시작과 종료 모두 약 43.5166px
- 상대 위치 오차: 0px
- 플레이어가 발판과 함께 이동한 거리: 약 127.07px

### 붕괴석 구조물 파괴 검사

- 실제 게임의 경로 충돌 함수를 사용해 붕괴석을 구조물 직전 경로값에 배치
- 파괴 구조물 그룹: 3개
- 감속 발생: 3회
- 파괴된 구조물은 플레이어 충돌 목록과 화면에서 함께 제거됨

### 최종 검사

- 자동 감사: 15/15 통과
- 압축 실행 파일 로드 성공
- 브라우저 예외·콘솔 오류: 0건

- 7차 완료 확인 폴링은 8차 활성화 뒤 중지

## 다음 차수

새 9차에서는 이번 8차의 세 층 스네이크 구조를 훨씬 길게 확장한다. 파괴 가능한 미로 구조물, 연속 벽타기, 급강하, 방향 반전, 이동 발판과 대시를 장거리 추격 중 끊김 없이 이어 붙이고, 붕괴석이 각 층의 외곽 통로를 따라 실제로 방향을 전환하며 추격하도록 구현한다.
