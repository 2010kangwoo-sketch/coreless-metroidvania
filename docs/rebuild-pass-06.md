# Coreless 새로운 40차 제작 — 6차

## 선행 확인: 5차 실제 화면

5차 터렛 회랑을 Chromium으로 다시 실행해 다음 문제를 확인했다.

- 천장 터렛이 천장 아래에 떨어져 있어 실제로 고정된 장치보다 공중에 떠 있는 것처럼 보였다.
- 오른쪽 벽 터렛도 벽과 포신 사이의 지지 구조가 약했다.
- 체크포인트 복귀 직후 무적 점멸 프레임에는 캐릭터가 완전히 사라져 화면에서 위치를 확인하기 어려웠다.

## 6차 선행 수정

- 천장 터렛마다 천장과 직접 이어지는 금속 지지대와 고정판을 추가했다.
- 오른쪽 벽 터렛에 벽 연결 지지대와 고정판을 추가했다.
- 터렛 몸체를 캐릭터와 비슷한 크기로 확대했다.
- 무적 시간에는 캐릭터를 숨기지 않고 반투명하게 표시하도록 변경했다.
- 기존 조준 예고, 엄폐, 포탄 밟기 기능은 그대로 유지했다.

## 전투 경기장 구현

- 공간 범위: x=9300~11700, 폭 2400px
- 전투 중 입구와 출구가 모두 잠기는 단일 경기장
- 전투 종료 후 오른쪽 출구 하나만 개방
- 일방통행 전투 발판 3개
- 입구·종료 체크포인트
- 세 파동 구성: 2기 → 2기 → 3기, 총 7기
- 파동 종료 시 체력 완전 회복

### 적의 크기

- 굴착충: 38×44px
- 기계 사냥개: 48×36px
- 부유 감시체: 40×34px
- 플레이어: 34×48px

모든 일반 적을 플레이어와 비슷하거나 약간 큰 범위로 제한했다.

### 전투 조작

- J 또는 X: 62px 근접 공격
- 공격 활성 시간: 12프레임
- 공격 재사용 시간: 22프레임
- 내려가면서 적의 상단에 닿으면 밟기 공격
- 밟기 성공 시 적에게 피해를 주고 플레이어는 위로 튕김
- 적에게 근접 공격이 적중하면 짧은 경직과 밀려남 적용

## 실제 Chromium 검사

### 실제 키보드 입력

- D 입력으로 x=9425px에서 약 9629.86px까지 이동하고 전투 시작 판정 확인
- J 키를 실제로 두 번 입력해 첫 번째 굴착충 1기 처치
- 적 위에서 낙하해 밟기 판정 증가 확인
- 경기장 정리 후 D 입력으로 오른쪽 출구를 통과해 `cp_arena_end` 도달
- 최종 x=11638px

### 결정적 전체 파동 검사

브라우저 내부의 동일한 전투·파동 코드를 사용하되, 남은 적의 체력을 테스트 훅으로 제거해 파동 전환과 문 잠금·해제를 반복 검증했다.

- 생성된 적: 7기
- 처치된 적: 7기
- 완료한 파동: 3개
- 종료 시 입구 잠금 해제 및 오른쪽 출구 개방
- 브라우저 콘솔 오류: 0건
- 자동 감사: 13/13 통과
