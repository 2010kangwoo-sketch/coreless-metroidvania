## 생성 프롬프트 세트

모든 자산은 `stylized-concept` 용도, 어두운 폐허형 기계 성소, 청록 주변광, 제한된 황동·녹·그을음, 평면 벡터와 깨끗한 SF 표면 금지를 공통 조건으로 사용했다.

1. 원경: 인물·플랫폼 없이 여러 겹의 거대한 기계식 볼트와 아치가 안개와 거리 감쇠 속으로 사라지는 넓은 2D 횡스크롤 배경. 차가운 청록 빛줄기와 소량의 먼 황색 점광만 사용한다.
2. 아치: 단일 부서진 원형 석조·기계 아치를 정측면으로 배치한다. 녹슨 철, 조각된 돌, 작은 황동 결합부를 사용하고 배경·그림자·문자·캐릭터는 넣지 않는다.
3. 기둥: 수직 제철소 지지 기둥 한 개를 정측면으로 배치한다. 산화 철, 조각석, 기계식 원형 관절과 소량의 황동을 사용하고 독립 실루엣을 유지한다.
4. 전경 잔해: 바닥을 따라 이어지는 무너진 기계 판재·석재·링 구조의 낮은 잔해 띠를 만든다. 중앙 상단은 플레이어 판독을 위해 비우고, 떠 있는 조각이나 배경 장면은 넣지 않는다.
5. 보행 거더: 이미지 폭 대부분을 차지하는 한 개의 연속 수평 보행 거더를 만든다. 걷는 윗면은 직선으로 유지하고, 낡은 돌·철 케이싱, 파이프, 리벳, 균열, 제한된 황동을 사용한다. 원근·난간·네온·끊어진 윗면은 금지한다.

투명 자산의 배경 지시는 `#00ff00` 단색, 그림자·그라데이션·바닥면·반사 없음, 피사체 내부에 키 색상 사용 금지로 고정했다.

## 판단과 30차 승인 관문

- 29차는 생성된 조각별 이미지 수준의 래스터 자산을 실제 게임의 독립 깊이층과 플레이 표면에 사용할 수 있음을 증명했다.
- 현재 고품질 적용 범위는 제철소 대표 구간이며 전체 초대형 방이 같은 품질로 교체된 상태는 아니다.
- 30차에서 이동 중 모듈 이음새, 충돌면과 표면 정렬, 전경 가림, 플레이어 대비, 평면 임시 그래픽 노출 면적을 대표 장면별로 판정한다.
- 30차 승인을 통과하기 전에는 이 화풍을 전 구역으로 대량 확장하지 않는다. 불합격 항목은 자산 재생성 또는 배치 재작업 후 다시 승인한다.
- 손상되지 않은 원본 최종 설계도 이미지가 다시 확보되면 30차 승인 전에 화면 구성·높이 흐름·비례를 직접 대조해야 한다.
