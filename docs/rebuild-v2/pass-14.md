# Rebuild V2 — 14차 거대 곡선 전환부

## 목표

13차 출구 `(19800, 7520)`에서 왼쪽 아래로 크게 휘어 내려간 뒤 자연 낙하하고, 진행 방향을 오른쪽으로 반전해 15차 낡은 다리 연결점 `(19000, 9000)`에 도달하는 추격 회색박스다.

- 상단 생존 경로: 왼쪽 이동, 120px 공백 점프, 점점 가팔라지는 경사
- 전환: 약 250px 자연 낙하 후 하단 경사 착지
- 하단 복귀 경로: 오른쪽 방향 반전 후 다리 연결점 도달
- 추격석: 플레이어 경로 아래의 420px 폭 전용 통로를 따라 같은 대곡선을 통과
- 카메라: 전환부에서 0.38배까지 축소해 상·하단 경로와 추격석을 함께 표시

## 구현 및 조정 이력

초기 집중 검사에서 추격석의 입구 교차, 13차 위험물의 잔존 판정, 출구에서 검사 입력이 다시 눌리는 문제가 드러났다. 13차 추격 종점을 플레이어 출구 아래 `(19800, 7640)`로 옮기고 100% 도달 뒤 14차를 시작하게 했으며, 14차 진입 후 이전 정밀 가시 판정을 비활성화했다.

첫 전체 무중단 검사는 1~13차와 14차 상단·낙하까지 통과했지만, 누적 추격 진행률이 집중 검사보다 약 0.3% 빨라 하단 복귀 바닥 `giant_curve_lower_roll_a`가 플레이어 아래에서 먼저 붕괴했다. 이 실패는 완주로 계산하지 않았다. 하단 복귀 바닥에 420px의 추가 후방 붕괴 간격을 적용한 뒤 전체 검사를 처음부터 다시 실행해 통과했다.

## 한계

- 현재는 물리·동선 검증용 회색박스이며 최종 지형 그래픽, 파괴 애니메이션, 음향과 전투는 없다.
