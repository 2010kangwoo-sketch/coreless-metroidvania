# Rebuild V2 — 15차 낡은 나무다리 피날레

## 목표

14차 연결점 `(19000, 9000)`에서 시작해 6,200px 길이의 우하향 낡은 다리를 건너고, 세 파손 간격과 마지막 공중 대시를 통과해 최종 석재 착지대 `(25200, 9600)`에 도달하는 추격 피날레 회색박스다.

- 플레이어: 다리 판재 11구간 → 140px 점프 → 160px 점프 → 180px 점프와 공중 대시 → 석재 착지대
- 추격석: 다리 아래 420px 통로를 따라가며 판재와 지지대를 뒤에서 파괴한 뒤 출구 아래로 낙하
- 완료 조건: 세 점프, 마지막 공중 대시, 최종 착지, 추격 진행률 0.995 이상과 추격석 낙하
- 10개 청사진 구역 모두 플레이 가능한 회색박스 충돌 구조로 연결

## 구현 및 수정 이력

집중 검사에서 다음 실패를 확인하고 모두 실패 이력으로 보존했다.

1. 첫 간격에서 `Space`를 10프레임에 놓아 점프 컷이 적용됐고 상승량이 부족했다. 일반 점프 유지 시간을 18프레임으로 변경했다.
2. 점프가 판재 끝보다 129px 앞에서 시작돼 상승 시간을 평지에서 소모했다. 이륙 조건을 판재 끝 35px 이내로 옮겼다.
3. 140px와 160px 간격은 통과했지만 마지막 180px 간격은 일반 점프로 약 9px 부족했다. 마지막 간격을 점프 후 공중 대시로 확정하고 완료 조건에도 포함했다.

첫 전체 무중단 검사는 15차 자체가 아니라 추격 속도 상속 문제로 실패했다. 피날레용으로 올린 최대 속도 6.45가 1차부터 적용되어 14차 하단 복귀에서 추격석 접촉이 발생했다. 이 실행은 성공으로 계산하지 않았다. 기존 1~14차 최대 속도 6.05를 유지하고 15차는 220프레임 선행 시간과 다리 내부 정지 지점만 사용하도록 수정한 뒤 전체 검사를 처음부터 다시 실행해 통과했다.

## 한계

- 현재는 충돌·동선·추격 페이싱을 검증하는 회색박스다. 최종 목재 텍스처, 로프, 먼지, 파편 애니메이션과 화면 연출은 후속 작업 대상이다.
- 추격석은 세계 하단 경계까지 내려가지만 최종 시네마틱이나 보스 전환은 없다.
- 전투, 적, 터렛, 피해, 음악과 효과음은 구현하지 않았다.
- 자동 입력 통과는 사람 손 기준 난이도와 재미를 확정하지 않는다.
