# Coreless Rebuild V2 — Pass 20

## 목적

19차 후속 붕괴 구간을 그대로 유지하면서, 추격·파괴 단계의 다음 이동 장치인 장거리 수평 방향 스프링을 추가했다. 플레이어는 달리는 방향을 유지한 채 스프링을 밟아 1,140px 낭떠러지를 한 번의 긴 포물선으로 건너고 낮은 회복 지점에 착지한다.

- 작업 브랜치: `rebuild/mega-room-40pass`
- Draft PR: #1
- 실행 버전: `rebuild-v2-pass20`
- `main` 병합: 없음

## 구현

`src/v2/pass20-level.js`에 19차 출구 뒤의 12구역 `CHASE SPRING FLIGHT`를 추가했다.

- 19차 출구 `(29400, 10520)`에서 20차 출구 `(33700, 11160)`로 이어지는 4,300px 하강 동선
- 폭 130px 방향 스프링 한 개와 `D` 방향 진입 조건
- 수평 속도 22, 수직 속도 -16.8의 발사와 68프레임 관성 보존
- 폭 1,140px 낭떠러지와 위험 바닥
- 발사·정점·낭떠러지 통과·착지·출구의 순차 완료 판정
- 비행 중 0.78배 광각 카메라와 착지 후 일반 카메라 복귀
- 스프링 궤적 안내선, 착지 분지, 출구 경사와 완료 표식
- 전체 세계 크기 35,000×12,200 및 청사진 12구역 확장

스프링은 플레이어가 지상에서 오른쪽으로 달리며 패드에 진입했을 때만 발동한다. 발사 직후 일반 공중 가속이 속도를 즉시 상쇄하지 않도록 68프레임 동안 수평 관성을 보존하되, 미세한 공중 제어와 감속은 유지했다.

## 집중 검사

집중 검사는 19차 완료 체크포인트 배치에만 좌표 도우미를 사용하고, 이후에는 실제 `D` 키 이벤트로 스프링 접근부터 출구까지 통과했다. 좌표 도우미를 썼으므로 전체 완주로 계산하지 않았다.

- 실제 입력: `D`
- 스프링 발사/착지: 1/1회
- 최고 수평 속도: 22
- 관성 보존: 68프레임
- 최종 위치: 약 `(33700.79, 11124.22)`
- 리셋/붕괴석 접촉: 0/0
- 런타임/20차 감사: 29/29, 36/36
- 콘솔/페이지 오류: 0/0
- 결과: 통과

최초 집중 실행은 서버리스 Chromium의 GPU 초기화 인자 차이로 브라우저가 시작되지 않았다. 전체 검사와 동일한 실행 인자를 적용해 해결했으며 게임 플레이 실패로 계산하지 않았다. 이후 시각 검사에서 비활성화된 18차 끝벽이 계속 그려지고 스프링이 바닥 뒤에 가려지는 렌더 순서를 발견해, 충돌 활성 상태와 렌더 상태를 일치시키고 패드를 바닥 위에 그리도록 수정한 뒤 다시 통과했다.

## 전체 검사 실패와 수정

전체 경로를 반복 실행하며 다음 실패를 완주와 분리해 기록했다.

1. 임시 브라우저 런타임이 사라진 상태와 Chromium 압축 해제 중 권한 오류로 두 차례 시작되지 않았다. 저장소를 바꾸지 않고 호환 런타임을 별도 경로에 복원했다.
2. 기존 검사기의 11차 출구 허용 범위가 넓어, 플레이어가 낮은 출구가 아니라 겹쳐 있던 10차 출구 경사 위에서 멈추고 붕괴석에 접촉했다. 목표 좌표와 세 번째 집게 해제 조건을 실제 낮은 출구에 맞췄다.
3. 조건 수정 뒤에도 전체 누적 상태에서는 세 번째 집게가 과거의 `double_wall_exit_rise` 경사와 충돌해 낮은 출구로 내려가지 못했다. 세 번째 집게를 사용한 뒤 이 과거 연결용 경사의 충돌과 렌더링을 함께 비활성화했다.

세 실행 모두 완주로 계산하지 않았다. 실제 충돌 수정은 11차 집중 검사에서 집게 3개와 낮은 출구 착지를 다시 확인한 뒤, 시작점부터 20차까지 세 번째 전체 검사로 검증했다.

## 남은 한계

- 스프링, 궤적 안내, 낭떠러지와 착지 분지는 이동·충돌 검증용 회색박스다.
- 현재 방향 스프링은 `D`를 유지한 진입을 전제로 하며 사람 손 기준 발동 범위와 속도 감각은 추가 조정이 필요하다.
- 발사와 착지에는 최종 애니메이션, 입자, 음향과 카메라 연출이 없다.
- 회복 지점은 구간 완료 판정이며 영구 저장 시스템과 연결되지 않았다.
- 자동 무중단 검사가 사람 플레이의 재미와 15~20분 목표 시간을 보장하지는 않는다.
