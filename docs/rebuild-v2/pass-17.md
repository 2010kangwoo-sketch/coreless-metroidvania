## 구현

`src/v2/pass17-art.js`에 17차 시각 데이터와 결정적 검사를 분리했다.

- 구역별 재질 프로필 10개
- 서로 다른 재질 종류 10개
- 표면 캡 두께와 재질 이음선 간격

재질 종류는 이끼 낀 석재, 젖은 조적, 매몰 석회암, 압축 셰일, 파손 청동, 산화 수로, 매달린 사암, 보강 철재, 심연 슬레이트, 풍화 목재로 구분했다.

## 실제 화면 수정 이력

첫 집중 검사는 수치상 11/11을 통과했지만 캡처에서 하강부 보강 프레임 일부가 공중에 떠 보였다. 자동 감사로 잡히지 않은 시각 품질 문제이므로 완료로 처리하지 않았다.

1. 구역 경계 비율로 배치한 장식을 청사진 경로에 고정했다.
2. 청사진의 거시 경로와 실제 세부 바닥이 다른 곳에 오차가 남아 있었다.
3. 최종적으로 모든 프레임·지지대·식생을 실제 충돌 바닥 선분의 중점에 고정했다.

세 번째 집중 검사에서 상부·산업 구간·하강부·피날레 네 화면의 구조물이 실제 표면과 연결되는 것을 확인했다.
