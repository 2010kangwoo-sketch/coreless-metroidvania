# Rebuild V2 — 13차 정밀 파쿠르

## 목표

13차는 12차 출구 `(20300, 7200)`에서 시작해 왼쪽 아래로 내려가며 짧은 점프와 긴 점프의 입력 강도를 구분하고, 낮은 천장 아래를 통과한 뒤 착지 직후 반대 방향으로 전환해 더 낮은 출구 `(19800, 7520)`에 도달하는 추격 회색박스다.

- 플레이어 흐름: 12차 출구 → 왼쪽 내리막 → 짧은 가시 간격 → 낮은 천장 → 왼쪽 하강 → 방향 전환 → 긴 가시 간격 → 오른쪽 출구
- 붕괴석 흐름: 같은 구조물 안의 420px 폭 파괴 통로를 따라 왼쪽 아래로 내려가 U자형 바닥을 돌고, 플레이어와 같은 출구 방향으로 다시 합류
- 필수 입력: `A` 이동 → 짧은 `Space` 입력과 조기 해제 → `D` 방향 전환 → 길게 유지한 `Space`
- 표현 단계: 충돌과 추격 동선 검증을 위한 회색박스이며 최종 지형 그래픽은 넣지 않음

- 기준선 좌표 도우미: 사용하지 않음
- 기준선 최종 위치: 약 `(20307.23, 7147.15)`
- 기준선 종료 프레임: 17,093
- 기준선 리셋/붕괴석 접촉: 0/0
- 기준선 콘솔/페이지 오류: 0/0

13차를 추가하기 전의 1~12차 실제 플레이에서는 새 기능 구현을 막는 실패가 발견되지 않았다. 따라서 기존 구간의 물리 수치를 임의로 바꾸지 않고 12차 출구 뒤에 13차 경로를 연결했다.

13차 초기 회색박스 검사에서는 다음 문제가 확인됐다.

1. 장식용 구조 프레임이 실제 충돌체처럼 동작해 플레이어가 옆으로 밀릴 수 있었다.
2. 경사 바닥의 중심점만으로 착지를 판정하면 캐릭터 발이 보이는 바닥에 닿았는데도 착지를 놓치는 경우가 있었다.
3. 짧은 점프 구간의 상단 가시 판정이 아래쪽 복귀 경로까지 겹칠 수 있었다.
4. 붕괴석이 신규 지지대에 도달하는 시간과 플레이어 정밀 점프 시간이 맞지 않아, 출구 도달 뒤 추격 완료를 지나치게 기다리는 경우가 있었다.

## 2. 이번에 수정한 내용

- 장식 프레임은 시각 요소로만 남기고 실제 이동 충돌에서는 제외했다.
- 13차 경사 바닥 착지는 캐릭터 중심점 하나가 아니라 발 너비와 바닥 구간의 겹침을 사용하도록 보완했다.
- 짧은 간격의 가시 판정을 상단 공백에만 한정해, 방향 전환 뒤 아래쪽 복귀 경로와 겹치지 않게 했다.
- 13차 구조 지지대 위치에 맞춰 붕괴석 정지 지점 3개와 파괴 지연 시간을 조정했다.
- 12차 출구에서 추격을 바로 봉인하지 않고 13차의 정밀 파쿠르와 붕괴석 합류가 끝난 뒤 봉인하도록 완료 시점을 연장했다.
- 추격 중 뒤쪽 방향 전환 바닥이 지지대 파괴와 함께 붕괴하도록 연결했다.

## 3. 새로 추가한 기능

### 짧은 점프와 긴 점프의 물리 구분

| 항목 | 값 |
|---|---:|
| 짧은 가시 간격 | 100px |
| 긴 가시 간격 | 195px |
| 낮은 천장 여유 | 72px |
| 조기 해제 점프 상승량 | 약 44.66px |
| 완전 유지 점프 상승량 | 약 114.89px |
| 조기 해제의 긴 간격 도달 거리 + 캐릭터 폭 | 약 185.02px |
| 완전 유지의 긴 간격 도달 거리 + 캐릭터 폭 | 약 257.10px |

짧은 간격은 `Space`를 빠르게 놓아야 낮은 천장에 부딪히지 않고 통과한다. 같은 조기 해제 입력은 195px 긴 간격에 약 10px 부족하고, 충분히 유지한 점프는 약 62px의 착지 여유가 있다. 완료 판정은 짧은 점프 이륙, 조기 해제, 낮은 천장 통과, 방향 전환, 긴 점프 이륙과 착지, 낮은 출구 도달을 순서대로 요구한다.

### 하나로 접힌 플레이어·붕괴석 경로

- 플레이어 경로 기준점: 10개
- 13차 플레이어 바닥: 6개, 모두 붕괴 가능
- 가시 공백: 2개, 아래 우회 바닥 없음
- 장식 구조 프레임: 5개
- 붕괴석 통로 기준점: 10개, 폭 420px
- 13차 신규 파괴 지지대: 5개
- 전체 추격 경로: 72점, 약 56,388.13px
- 13차 경로 연장: 약 2,665.38px
- 전체 붕괴 패널: 61개
- 전체 파괴 지지대: 42개

붕괴석은 순간 이동하거나 별도 가이드 선으로 공중 이동하지 않는다. 외부 경사와 구조물 안에 표시된 넓은 통로 중심을 따라 내려가며, 왼쪽 최저점에서 완만한 U자형 경로로 방향을 바꿔 플레이어 출구 쪽으로 다시 올라온다. 두 가시 구간은 플레이어에게만 위험하며 붕괴석은 그대로 통과한다.

## 6. 아직 남은 한계

- 1차 시작점부터 13차 출구까지의 전체 무중단 키보드 완주는 이번 차수에서 별도로 성공시키지 않았다. 이전 구간 1~12차 기준선과 12차 출구→13차 출구 집중 검사를 나누어 수행했다.
- 현재는 물리와 동선을 검증하는 회색박스다. 장식 프레임, 파편, 먼지와 화면 흔들림은 임시 표현이며 지형 그래픽과 배경 3단계는 후속 차수 대상이다.
- 13차의 방향 전환은 정밀 파쿠르용 소형 U자 구조다. 14차에서 요구하는 거대한 원호형 하강 전환부는 아직 구현하지 않았다.
- 10구역 낡은 나무다리 피날레는 아직 청사진 데이터이며 15차 구현 대상이다.
- 전투, 적, 터렛, 피해, 음악, 효과음과 시네마틱 카메라는 이번 차수 범위가 아니다.
- 자동 입력 통과는 사람 기준 난이도와 재미를 확정하지 않는다. 최종 난이도 조정 때 입력 여유와 화면 가독성을 다시 검사해야 한다.
