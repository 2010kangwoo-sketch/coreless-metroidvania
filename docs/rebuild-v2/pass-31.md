## 구현

### 두 장면 레지스트리

1. `start_slope_sanctuary`
   - 시작 평지, 정원 경사, 부서진 하강 경사
   - 거대 매몰 성소와 비대칭 뿌리 아치를 중심으로 구성
2. `double_wall_shaft`
   - 첫 낙하, 두 벽 점프, 두 번째 수직 하강, 3구역 연결 선반
   - 개방형 리프트 탑과 수직 기계 성소를 중심으로 구성

카메라 X `2380~2620`의 240px 구간에서는 두 장면의 배경·중경·전경을 교차 페이드한다. 따라서 장면 경계에서 배경이 한 프레임에 교체되지 않는다.

원경은 WebP 불투명 플레이트로, 나머지 네 에셋은 투명 PNG 모듈로 사용한다. 모든 투명 에셋의 네 모서리 알파는 0이고 선명 크로마키 녹색 잔존 픽셀은 0개다.

### 충돌 정렬 표면

- 기존 `collisionFloors`와 `collisionSolids` 배열을 그대로 사용한다.
- 경사 표면은 각 충돌선의 길이와 각도로 `route-stone`을 회전·스케일한다.
- 표면 상단 오프셋은 `-3px`로 제한한다.
- 벽 점프용 수직 고체는 같은 에셋을 90도 회전해 실제 좌우 충돌면을 표시한다.
- 장면 에셋이 로드되지 않으면 기존 Canvas 지형이 그대로 남는 폴백을 유지한다.

## 이미지 생성 프롬프트 세트

31차 에셋은 프로젝트의 기존 어두운 매몰 기계 성소 방향에 맞춰 다음 프롬프트로 생성한 뒤 크로마 제거와 투명 가장자리 감사를 거쳤다.

1. 원경: 거대한 매몰 입구 정원 성소, 먼 조각 볼트·부서진 수로·수직 기계·뿌리·안개, 차가운 청록 광선과 절제된 호박색, 전경 플랫폼과 플레이어 제외, 평면 벡터와 깨끗한 SF 제외.
2. 아치: 침식된 석재·산화철·황동으로 된 하나의 비대칭 첨두 성소 아치, 오른쪽 상단 파손과 왼쪽 뿌리 기단, 녹색 크로마 배경, 플랫폼 제외.
3. 샤프트: 개방형 벽 점프 베이·가로보·도르래·사슬·버팀 기단을 가진 매우 높은 폐허 리프트 탑, 녹색 크로마 배경, 수평 플랫폼 제외.
4. 전경: 양쪽은 조밀하고 중앙 플레이 공간은 열린 낮은 석조·기계 파편·뿌리 스트립, 녹색 크로마 배경.
5. 경로 표면: 끊기지 않은 직선 상단을 가진 긴 뿌리 석재·철제 보행면, 산화 리브와 황동 세부, 파운드리 파이프 디자인 제외, 녹색 크로마 배경.
