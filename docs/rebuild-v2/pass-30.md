## 검사 중 발견하고 수정한 불합격

### 원경 오른쪽 잠재 이음새

29차 원경 판은 기준 화면에서는 충분했지만 실제 이동 범위 오른쪽 끝에서 약 66px가 부족할 수 있었다. 원경 목적지를 `1450×780`으로 확장하고 위치를 `(−80, −48)`로 보정했다. 수정 후 카메라 5개 표본의 좌·우·상·하 빈틈은 모두 0px다.

### 왼쪽 플레이 표면 클립

첫 집중 화면의 수치는 통과했지만 직접 검토에서 낮은 오르막 플레이어 아래 거더가 비어 보였다. 원인은 28차 백플레이트 경계 `x=16480`이 29차 모듈 장면의 거더에도 남아, 실제 플레이어 `x=16400`보다 80px 오른쪽에서 렌더링이 시작된 것이었다.

모듈 장면의 승인 플레이 표면 범위를 `(16300, 2700, 1950, 1350)`으로 분리하고, 다섯 표본의 플레이어 발 좌표가 모두 이 범위 안에 들어오는 검사를 추가했다. 보정 후 첫 화면에서 플레이어 발과 경사 거더가 연속적으로 맞물리는 것을 직접 확인했다.

## 승인 판단과 한계

- 28차는 래스터 품질 가능성을 증명했고, 29차는 이를 재사용 가능한 깊이 모듈과 충돌 정렬 표면으로 분리했으며, 30차는 대표 제철소 구간이 실제 이동 화면에서도 승인 기준을 충족함을 확인했다.
- 이 승인은 동일 파이프라인을 나머지 경로에 확장해도 된다는 의미다. 현재 전체 초대형 방의 모든 구역이 최종 래스터 미술로 교체됐다는 의미는 아니다.
- 나머지 경로에는 여전히 Canvas 기반 임시 지형과 구조물이 존재한다. 31차부터 승인된 미술 언어와 측정 기준을 사용해 구역군별로 래스터 적용 범위를 확장해야 한다.
- 손상되지 않은 원본 최종 설계도 이미지를 다시 확보하기 전까지 원본과의 픽셀·비례 직접 비교는 보류 상태다. 따라서 원본과의 정확한 일치까지 승인한 것으로 기록하지 않는다.
